Bisq Connect for iOS

EU Install via AltStore PAL

⚠️ Coming Soon: This distribution method is pending Apple notarization approval.

For users in the European Union, Bisq Connect will be available through AltStore PAL using Apple's alternative distribution.

Prerequisites

  • iPhone with iOS 17.4 or later
  • Device region set to an EU country
  • AltStore PAL installed (€1.50/year)

Installation Steps

  1. Open AltStore PAL on your iPhone
  2. Go to the Browse tab
  3. Tap Sources in the top-left
  4. Tap the + button and enter this URL:

https://bisq-network.github.io/bisq-mobile/apps.json

  1. Find "Bisq Mobile" in your sources
  2. Tap Get next to Bisq Connect
✓ No expiration! Apps installed via AltStore PAL don't require weekly re-signing.

Advanced Non-EU Sideloading

For users outside the EU, you can sideload the app using the free version of AltStore. This requires a computer and weekly re-signing.

⚠️ Limitations: Apps expire after 7 days and must be refreshed via AltStore connected to your computer.

Steps

  1. Install AltStore (free) and AltServer on your computer
  2. Download the IPA file from GitHub Releases
  3. Open the IPA file with AltStore to sign and install
  4. Refresh the app every 7 days via AltStore

Requirements

  • iOS Version: 16.0 or later
  • Bisq Node: A running Bisq 2 node (headless or Bisq2 Desktop) to connect to
  • Network: Local network or Tor access to your node (for more information visit Bisq Connect Wiki Tutorial)
